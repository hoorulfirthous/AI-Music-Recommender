#  MoodTune AI – Smart Music Recommendation System
##  Overview
MoodTune AI is a Streamlit-based web application that recommends songs based on the user's mood.
It uses simple Natural Language Processing (NLP) to analyze user input and suggest relevant songs.

##  Features

*  Mood detection from text input
*  Music recommendations based on mood
*  YouTube integration for listening
*  Smart keyword-based NLP system
*  Clean and interactive UI using Streamlit

##  Tech Stack

* Python 
* Streamlit 
* Basic NLP (rule-based)
* YouTube search integration

##  How It Works

1. User enters their mood in natural language
2. NLP logic detects mood using keyword scoring
3. Songs are fetched based on detected mood
4. User can listen via YouTube links

##  Future Enhancements

*  Voice input for mood detection
*  Favorites system using database
*  Spotify API integration
*  Advanced AI/ML mood detection
